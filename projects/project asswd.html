In 2023, as part of the Advanced Server-side Web Development module, I completed a MERN stack project. The project involved developing a dynamic web application using MongoDB, Express.js, React, and Node.js. Utilizing the MERN stack, I built a robust and scalable application with a powerful backend powered by Node.js and Express.js, which provided efficient server-side processing and RESTful APIs. The front end was developed using React, enabling a seamless and responsive user interface. I implemented efficient data storage and retrieval with MongoDB as the database, ensuring optimal performance and data integrity. Overall, the MERN stack project showcased my ability to develop full-stack web applications, combining powerful technologies for a seamless user experience.
